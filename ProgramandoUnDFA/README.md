## Tipos de Tokens
* Enteros
* Flotantes (Reales)
* Operadores:
  * Asignación
  * Suma
  * Resta
  * Multiplicación
  * División
  * Potencia
* Identificadores:
  * Variables
* Símbolos especiales:
  * (
  * )
* Comentarios:
  * // seguido de caracteres hasta que se acabe el renglón
